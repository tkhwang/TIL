### Scaling 개선 노력

* Casper : PoW 방식을 PoS 방식으로.
* Sharding : 샤드를 통한 2계층 블록체인
* Raden : 상태 채널을 통해 이더 및 ERC20과 호환되는 토큰을 송금할 수 있는 기능. 사용자간 처음과 마지막 거래 사실만 블록체인에 기록
* Plasma : 트리 구조의 off-chain 다중 블록체인
* LES : Light 싱크를 통한 블록체인 경량 동기화
* DAICO : 탈중앙화 ICO
